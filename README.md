## Run Locally   

Clone the project

``` console
git clone git@github.com:Harrytimbog/TypeScript-With-NodeJS.git
```

Go to the project directory

``` console
cd TypeScript-With-NodeJS
```

Install dependencies

``` console
npm install
```

Install NPM Packages

``` console
npm install
```

Start Server

``` console
npm start
```
